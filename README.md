# CoolLensCV
coollenscv is a creative image processing project that adds stylish sunglasses to passport-size photos using face detection techniques.

## Overview
Detects faces in an image and accurately overlays sunglasses, turning simple photos into unique, styled outputs.
## Features
Detects faces in an image using computer vision techniques
Accurately positions a sunglasses overlay on the detected face
Optimized for individual passport-size photos
Easily customizable for different sunglasses styles and image types
## Technologies Used
Python
OpenCV – for image processing and face detection

NumPy – for numerical operations and array handling
## How to Use
1)Load your photo
2)Detect the face
3)Find the eye area
4)Fit the sunglasses to that area
5)Place it on the face
